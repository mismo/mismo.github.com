JavaScript Pattern Matching with Regular Expressions


A number of punctuation characters have special meanings in regular expressions. They are:(在正则表达式中，许多标点符号具有特殊的含义。它们是：)
^ $ . * + ? = ! : | \ / ( ) [ ] { }

2、Regular expression character classes(正则表达式的字符类)

Character
字符		Matches
匹配
[...]Any one character between the brackets.
位于括号之内的任意字符
[^...]Any one character not between the brackets.

.Any character except newline or another Unicode line terminator.

3、Regular expression repetition characters(正则表达式的重复字符)

Character
字符		Meaning
含义
{n,m}Match the previous item at least n times but no more than m times.
匹配前一项至少n次，但是不能超过m次
{n,}Match the previous item n or more times.
匹配前一项n次，或更多次
{n}Match exactly n occurrences of the previous item.
匹配前一项恰好n次
?Match zero or one occurrences of the previous item. That is, the previous item is optional. Equivalent to {0,1}.
匹配前一项0次或1次，也就是说前一项是可选。等价于{0,1}
+Match one or more occurrences of the previous item. Equivalent to {1,}.
匹配前一项1次或多次，等价于{1,}
*Match zero or more occurrences of the previous item. Equivalent to {0,}.
匹配前一项0次或多次，等价于{0,}

4、Regular expression alternation, grouping, and reference characters(正则表达式的选择、分组和引用字符)

Character
字符		Meaning
含义
|Alternation. Match either the subexpression to the left or the subexpression to the right.
选择。匹配的是该符号左边的子表达式或右边的子表达式
(...)Grouping. Group items into a single unit that can be used with *, +, ?, |, and so on. Also remember the characters that match this group for use with later references.
组合。将几个项目组合为一个单元，这个单元可由*、+、?和|等符号使用，而且还可以记住和这个组合匹配的字符以供此后的引用使用
(?:...)Grouping only. Group items into a single unit, but do not remember the characters that match this group.
只组合。把项目组合到一个单元，但是不记忆与该组匹配的字符
\nMatch the same characters that were matched when group number n was first matched. Groups are subexpressions within (possibly nested) parentheses. Group numbers are assigned by counting left parentheses from left to right. Groups formed with (?: are not numbered.
和第n个分组第一次匹配的字符相匹配。组是括号中的子表达式(可能是嵌套的)。组号是从左到右计数的左括号数，以(?:形式分组的组不编码

5、Regular-expression anchor characters(正则表达式的锚字符)

Character
字符		Meaning
含义
^Match the beginning of the string and, in multiline searches, the beginning of a line.
匹配字符串的开头，在多行检索中，匹配一行的开头
$Match the end of the string and, in multiline searches, the end of a line.
匹配字符串的结尾，在多行检索中，匹配一行的结尾
\bMatch a word boundary. That is, match the position between a \w character and a \W character or between a \w character and the beginning or end of a string. (Note, however, that [\b] matches backspace.)
匹配一个词语的边界。简而言之，就是位于字符\w和\W之间的位置，或位于字符\w和字符串的开头或结尾之间的位置(但注意，[\b]匹配的是退格符)
\BMatch a position that is not a word boundary.
匹配非词语边界的位置
(?=p)A positive lookahead assertion. Require that the following characters match the pattern p, but do not include those characters in the match.
正前向声明，要求接下来的字符都与模式p匹配，但是不包括匹配中的那些字符
(?!=p)A negative lookahead assertion. Require that the following characters do not match the pattern p.
反前向声明，要求接下来的字符不与模式p匹配

6、Regular-expression flags(正则表达式的标志)

Character
字符		Meaning
含义
iPerform case-insensitive matching.
执行不区分大小写的匹配
gPerform a global matchthat is, find all matches rather than stopping after the first match.
执行一个全局匹配。简而言之，即找到所有匹配，而不是在找到第一个之后就停止
mMultiline mode. ^ matches beginning of line or beginning of string, and $ matches end of line or end of string.
多行模式，^匹配一行的开头或字符串的开头，$匹配一行的结尾或字符串的结尾
